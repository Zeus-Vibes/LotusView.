# Image Assets

Place your images in this directory with the following names:

## Hero Slider Images
- `hero1.jpg` - First slide image (1920×1080px recommended, 16:9 aspect ratio)
- `hero2.jpg` - Second slide image (1920×1080px recommended)
- `hero3.jpg` - Third slide image (1920×1080px recommended)

## About Section
- `about.jpg` - About section image (600×600px or 800×600px recommended)

## Portfolio Section (6 images)
- `p1.jpg` - Portfolio item 1 (400×400px square recommended)
- `p2.jpg` - Portfolio item 2 (400×400px square recommended)
- `p3.jpg` - Portfolio item 3 (400×400px square recommended)
- `p4.jpg` - Portfolio item 4 (400×400px square recommended)
- `p5.jpg` - Portfolio item 5 (400×400px square recommended)
- `p6.jpg` - Portfolio item 6 (400×400px square recommended)

## Image Optimization Tips

1. **Compress Images**: Use TinyPNG, ImageOptim, or similar tools
2. **Format**: Use JPEG for photos, PNG for graphics with transparency
3. **Resolution**: Target file sizes 100-300 KB per image
4. **Dimensions**: Follow recommendations above
5. **Web Format**: Consider WebP format for better compression (with fallbacks)
